# 台股研究 Quickstart

## 1) 先設定角色
把 `playbooks/tw-stock/system_prompt.md` 內容貼到 OpenClaw 的 system prompt。

## 2) 每日任務起手 prompt
可直接貼以下內容：

```text
請依照台股研究模板輸出今日報告。
範圍：0050 成分股 + 今天成交量前 30 名。
輸出：市場快照、3-5 檔觀察清單、反方風險、明日追蹤指標、來源與日期。
注意：結論都要有資料日期與來源，若資料不足請明確標示。
```

## 3) 建議先從「研究」開始，不直接下單
- 第一週：只驗證報告品質與穩定性
- 第二週：加入自選股與產業題材
- 第三週：再建立自己的進出場規則（由你決策）

## 4) 使用注意
- 這是研究輔助，不是投資建議。
- 關鍵數字請在下決策前再手動複核一次。
